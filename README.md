# Fussballiga Simulator 1. Bundesliga

## Projektziel

Ziel dieses Projekts ist die Entwicklung einer Konsolenanwendung, die die 1. Bundesliga simuliert. Die 18 Bundesliga-Mannschaften sind im System hinterlegt und treten in einer vollständigen Saison mit Hin und Rückrunde gegeneinander an. Das System simuliert alle Spiele automatisch und gibt abschliessend die Abschlusstabelle mit Punkten, Toren und Tordifferenz im Terminal aus.
